# Are-there-induction-heads-
Code for producing both dataset and figures used in the toy dataset post "Are there induction heads in transformers?"

### Setup
To run the repository, ensure a Python version >= 3.10.11, and run
```
pip install -r requirements.txt
```

Functions for creating samples of the dataset are modular and provided in the **create_dataset.py**-file. To run the entire pipeline and produce the graphs in the associated post, run: 
```
python main.py
```
